# Общие текстуры

Эта папка хранит общие текстуры, которые синхронизируются в локальный рабочий каталог `_source/_shared_textures`.

Правила:
- Поддерживаемые форматы (LFS): png/jpg/jpeg/tga/dds/tif/tiff/exr/hdr.
- Текстовые заметки и плейсхолдеры можно хранить как .txt.
- Добавляйте новые текстуры только в эту папку, чтобы их могли получить все.
